Welcome to the Script Runner

This application is designed to help you run scripts with ease. Here's a brief overview of how to use it:

  • Input File: If the script requires an input file, browse and select the file using the 'Browse' button.
  • Output File: If your script generates an output file, specify the desired location of the output file.

Once you've filled out all the necessary fields, click the 'Run' button to execute the script.

If you encounter any errors while running the script, please ensure that all required fields are filled out correctly. If the problem persists, email lucashart@mines.edu for further assistance.
