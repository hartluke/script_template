Welcome to the Script Runner

This application is designed to help you run scripts with ease. Here's a brief overview of how to use it:

  • Input File: If your script requires an input file, browse and select the file using the 'Browse' button.
  • Output File: If your script generates an output file, specify the desired location and name of the output file.
  • Run Mode: Choose the mode in which you want to run the script. The options are 'Production' and 'SB'.

Once you've filled out all the necessary fields, click the 'Run' button to execute the script. You can monitor the progress of the script through the loading dialog that appears.

If you wish to terminate the application at any point, simply click the 'Exit' button.
